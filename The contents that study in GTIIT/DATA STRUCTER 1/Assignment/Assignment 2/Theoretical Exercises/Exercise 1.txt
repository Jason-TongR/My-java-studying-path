Exercise 1 : A data structure for select and rank



(a)    
    public class RankSelect{
        
        private ArrayList<Integer> number_of_true;          //in the field of our data structure , we have the number of true array , the element is the rank(A,i) in i position 
        
        public RankSelect( ArrayList<Boolean> A ){          //this is the constructor , initializes the data structure

            number_of_true = new ArrayList<>();          
            
            int count_of_one = 0;

            for( int i = 0 ; i < A.size() ; i++ ){
                
                number_of_true.add(count_of_one);

                if( A.get(i) )
                    count_of_one++;

            }

            number_of_true.add(count_of_one);

        }



        public int rank(int i){

            assert ( i >= 0 ) && ( i < number_of_true.size() ) : "precondition , since the size of A = the size of number_of_true  , so 0 <= i < n = size of A "

            return number_of_true.get(i);
        }

    }

    
    COMMENT :   1. In the initializes part of our data structure (i.e the constructor part) , the Worst time complexity is O(n) , because we have only one for loop in size n
                
                2. in the rank(int i) , the Worst time complexity is O(1) , because all code in this method take O(1) .

            in Exercise (a) , we know that the method rank( int i ) is to return the number of "true" in the array A[0,1,...,i-1]
            
            and we note that the outputs of the method rank(int i) actually are monotonic non-decreasing

            so , actually , select(j) is the smallest index k such that rank(k + 1) == j + 1.

            that is , we want to find that k.

            More , since the output of the rank operation is monotonic non-decreasing ,

            so we can do binary search over the possible index range [0, n-1] to find that k.

            we know that the Worst case of the binary search is O(log n) , the time complexity of rank( int i ) is O(1) . So the total Worst time complexity is O(log n). DONE! 

            Therefore , we proved that just by using rank operation , and without storing additional information in the data strucure , we can implement the select( int j ) operaiton.
